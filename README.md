# COSEAL Best Practices

This is the best practices for research paper from the COSEAL community. It is a living document and we welcome your input!

## Discussions
As best practices are evolving, we're happy for discussions on their validity emerge. The best place to do this are the issues of this repository. 
Simply create a new issue and describe the changes you'd like to see or the opinions you disagree with. The COSEAL chairs will then facilitate the discussion to improve the paper.

## Contributions
If you would like to directly contribute to the paper, you can open a PR the general chairs will then review. Please include the following information:
1. A short changelog
2. The motivation for your changes
3. Have you included yourself as a co-author? This should happen for most changes
